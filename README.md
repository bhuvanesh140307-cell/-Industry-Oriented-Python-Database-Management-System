# -Industry-Oriented-Python-Database-Management-System# Industry-Oriented Python Database Management System

## Problem Statement

Many business applications require interaction with databases for storing, retrieving, and updating data. Manual database handling can lead to data inconsistency and performance issues. Python provides a reliable way to build database-driven applications.

## Objective

To develop a Python-based application that performs efficient database operations for managing employee records.

## Features

* Database Schema Design
* Python Database Connectivity
* CRUD Operations (Create, Read, Update, Delete)
* Data Validation
* Error Handling
* Command Line Interface

## Technologies Used

* Python
* SQLite3
* VS Code
* GitHub

## Database Schema

Employee Table:

| Field      | Data Type           |
| ---------- | ------------------- |
| id         | INTEGER PRIMARY KEY |
| name       | TEXT                |
| department | TEXT                |
| salary     | REAL                |
